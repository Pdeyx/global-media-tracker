Global Media Tracker

Full-Stack аналитический дашборд. Мониторинг предвзятости СМИ в реальном времени с использованием алгоритмов NLP и нейросети Llama 3.

💡 Архитектура проекта

Этот проект демонстрирует полный цикл работы с данными (Data Engineering):

  • Backend (Python/FastAPI): REST API с интеграцией Google News RSS.
  • Data Science (VADER & Llama 3): Гибридный анализ тональности в реальном времени (Real-Time AI Agents).
  • База данных: Локальное хранилище на SQLite с защитой от дубликатов.
